## Calculations in bank
### Example for usage:
`python main.py --part=0.2 --val=60 --periods=4`<br>
`
+---------+--------------------------+------------------------------------------+
| Periods | Tatal sum increasing 0.2 | Profit you achieve if you save the money |
+---------+--------------------------+------------------------------------------+
|    0    |         60.0000          |                  0.0000                  |
|    1    |         72.0000          |                 12.0000                  |
|    2    |         86.4000          |                 14.4000                  |
|    3    |         103.6800         |                 17.2800                  |
|    4    |         124.4160         |                 20.7360                  |
+---------+--------------------------+------------------------------------------+
`
### Installing requirements:
`pip install -r requirements.txt`
### See help:
`python main.py --help`
